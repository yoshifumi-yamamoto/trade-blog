## EUR/USD

### 日足

![EUR/USD_D1](/img/2020-11-17-eurusd-d1-tradexfin-limited.png)

POの形成はなく方向感はありません。

<br/>
<br/>

### 4時間足

![EUR/USD_H4](/img/2020-11-17-eurusd-h4-tradexfin-limited.png)

上昇のPOを形成しています。安値を切り上げて上昇して来ていますので、このまま1.19000代に突入する可能性が出て来ました。

<br/>
<br/>

### 1時間足

![EUR/USD_H1](/img/2020-11-17-eurusd-h1-tradexfin-limited.png)

POは形成していませんが、EMA75付近がサポートとなって上昇トレンドを形成しています。

<br/>
<br/>

### 考察＆戦略

方向感が分かりにくい相場ですが、中期と短期で買いが優勢になっています。直近高値の1.18950付近までは目指す流れです。1.19000からは売りが出やすいのでブレイクしたとしてもロングは狙いにくいので、エントリーが難しい相場状況となっています。

<br/>
<br/>

## EUR/JPY

### 日足

![EUR/JPY_D1](/img/2020-11-17-eurjpy-d1-tradexfin-limited.png)

POの形成はなく方向感はありません。

<br/>
<br/>

### 4時間足

![EUR/JPY_H４](/img/2020-11-17-eurjpy-h4-tradexfin-limited.png)

緩やかにですが、下降のPOを形成しています。陰線と陽線が入り乱れて揉み合っているようにも見えます。調整の下降の可能性が高いです。

<br/>
<br/>

### 1時間足

![EUR/JPY_H1](/img/2020-11-17-eurjpy-h1-tradexfin-limited.png)

POの形成はありませんが、緩やかに下降トレンドを形成しています。EMA200付近がサポートとして機能しています。

<br/>
<br/>

### 考察＆戦略

長期では方向感はありませんが、中期と短期で買いが優勢の状況です。急上昇後の調整の下降に入っているのですが、買い上がりそうで中々買いあがらない難しい相場状況です。どこが押し目となるか分からないので、ショートでは狙わない方が良いですね。ロングを打つ場合は124.500を超えてから、もしくは125.000をブレイクしてからが安全です。

<br/>
<br/>

## AUD/USD

### 日足

![AUD/USD_D1](/img/2020-11-17-audusd-d1-tradexfin-limited.png)

角度が緩やかですが上昇のPOを形成しています。MA5を一度割り込みましたが、抜け直しの形となりました。

<br/>
<br/>

### 4時間足

![AUD/USD_H4](/img/2020-11-17-audusd-h4-tradexfin-limited.png)

まだ、POの形成とまではいきませんが安値を切り上げて上昇して来ています。これから上昇のPOを形成してくるかというところで、直近の高値付近まで到達しています。

<br/>
<br/>

### 1時間足

![AUD/USD_H1](/img/2020-11-17-audusd-h1-tradexfin-limited.png)

上昇のPOを形成しています。そこまでボラティリティは大きくありませんが、大きな売りが入ることもなく上昇して来ています。

<br/>
<br/>

### 考察＆戦略

全体的に買いが優勢の状況です。ロングを狙っていきます。直近高値の0.73400付近をブレイクしたところでロングを狙っていきます。0.74000付近が次のターゲットになります。ここは日足レベルの前回高値ですが、安値を切り上げて上昇して来ていますので、ブレイクの可能性があります。ポジションを保有した場合は反応を見ながら決済を考えていきます。

<br/>
<br/>

## USD/CAD

### 日足

![USD/CAD_D1](/img/2020-11-17-usdcad-d1-tradexfin-limited.png)

POの形成はありませんが、下降トレンドを形成しています。これから下降のPOを形成してくるかといったところです。

<br/>
<br/>

### 4時間足

![USD/CAD_H４](/img/2020-11-17-usdcad-h4-tradexfin-limited.png)

上昇のPOを形成していましたが、EMA200がレジスタンスとなり売りに押されています。このまま前回安値を目指す流れです。

<br/>
<br/>

### 1時間足

![USD/CAD_H1](/img/2020-11-17-usdcad-h1-tradexfin-limited.png)

緩やかですが、下降のPOを形成しています。一度POが崩れかけましたが、EMA200付近がレジスタンスとなって再度下降して来ています。

<br/>
<br/>

### 考察＆戦略

全体的に売りが優勢の相場です。ショートを狙っていきます。現在１時間足レベルで下降のPOを形成していますので、このまま流れに乗っていきます。直近安値の1.3650付近をブレイクしたところでショートを狙います。出口は反応を見ながらですが、1.29250付近まで狙います。
